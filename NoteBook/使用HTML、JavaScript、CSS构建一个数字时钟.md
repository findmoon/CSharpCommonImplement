## 数字时钟项目的文件结构

创建一个包含 HTML、CSS 和 JavaScript 文件的根文件夹。可以将文件命名为任何你想要的名称。

这里的根文件夹名为`Digital-Clock`，根据标准命名约定，HTML、CSS 和 JavaScript 文件分别命名为 `index.html`、`styles.css` 和 `script.js`。
