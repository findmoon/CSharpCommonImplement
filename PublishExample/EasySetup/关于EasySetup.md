
# 介绍

这是关于EasySetup能找的最新版本，似乎相关软件的开发和维护已经停止，官网中也已经找不到改软件。

比如，在腾讯软件中心中，能找到该软件。【3.14】

官网为：http://bersoft.com/  里面已经没有了相关介绍【在程序运行界面、腾讯软件中心、国外的一个网站中，都记录此地址为EasySetup的官网】

EasySetup的安装使用非常简单，基本打开就能用（每个步骤、功能、菜单都很明确，界面中的内容也可以直接编辑，几乎不需要相关教程）。

唯一的问题就是 首屏(Screen1)中的 URL 不能修改，只能保持默认的`http://bersoft.com`，要修改需要 **注册**。【从菜单中找到注册地址，也是不可用的。】

# 如何对EasySetup的注册进行破解？

由于没有了官网，并且网上找不到相关的破解软件【很多都是低版本的EasySetup】。目前能想到的唯一可以注册的可行方式就是“破解”。

可是，自己的能力又不足以破解【似乎进行破解时，为了防止可能的意外，最好在单独的虚拟机中进行(防止格式化)】。

以后待了解。
